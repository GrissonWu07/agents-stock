# 前端页面与交互清单

本文档只描述当前 `ui/` SPA 已经存在的页面、主要快照接口和关键交互，不再使用旧的“量化候选池 / continue 回放”口径。

## 1. 当前路由

| 路由 | 页面 | 主要快照接口 |
|---|---|---|
| `/main` | 工作台 | `/api/v1/workbench` |
| `/discover` | 发现股票 | `/api/v1/discover` |
| `/research` | 研究情报 | `/api/v1/research` |
| `/portfolio` | 持仓诊断 | `/api/v1/portfolio_v2` |
| `/portfolio/position/:symbol` | 个股详情 | `/api/v1/portfolio_v2/positions/{symbol}` |
| `/live-sim` | 实时量化 | `/api/v1/quant/live-sim` |
| `/his-replay` | 历史回放 | `/api/v1/quant/his-replay` |
| `/ai-monitor` | AI 盯盘 | `/api/v1/monitor/ai` |
| `/history` | 历史记录 | `/api/v1/history` |
| `/settings` | 环境配置 | `/api/v1/settings` |

## 2. 工作台 `/main`

### 页面定位

工作台当前展示的是统一股票池视图，不只是纯 `watched` 股票。

### 当前主要区域

1. 概览卡
2. 股票池表格
3. 股票分析区
4. 下一步入口
5. 最近活动

### 股票池表格的当前动作

- 手工添加股票
- 批量加入实时量化
- 批量登记持仓
- 批量删除
- 行内查看个股详情

说明：

- `refresh-watchlist` 的旧按钮语义已经停用，后端会报错提示使用统一数据刷新链路。

## 3. 发现股票 `/discover`

### 页面定位

发现股票页负责生成候选结果，并把股票推进到统一股票池。

### 当前主要动作

- `运行策略`
- `批量加入关注池`
- 行内 `加入关注池`
- `重置列表`

### 当前结果来源

- `selector_results/*.json`
- `stock_runtime_snapshot.json` 叠加实时名称和价格

## 4. 研究情报 `/research`

### 页面定位

研究情报页展示板块、龙虎榜、新闻和宏观结果；只有存在明确股票输出时，才提供写入股票池的动作。

### 当前主要动作

- `重新生成`
- `批量加入关注池`
- 行内 `加入关注池`
- `重置列表`

## 5. 持仓诊断 `/portfolio`

### 页面定位

当前前端已经使用 `portfolio_v2` API 族，不再以旧 `portfolio_stocks.db` 为前提写文案。

### 当前主要能力

- 组合概览
- 当前登记持仓列表
- 个股详情跳转
- 刷新组合
- 保存/启动/停止调度
- 刷新技术指标
- 更新持仓
- 删除持仓

## 6. 实时量化 `/live-sim`

### 页面定位

UI 文案已经改成：

- `实时量化股票`

### 当前主要区域

1. 左栏配置和状态
2. 账户概览
3. 实时量化股票表
4. 执行中心
5. 当前持仓
6. 成交记录
7. 信号表

### 当前主要动作

- `保存`
- `启动模拟`
- `停止模拟`
- `重置`
- `批量量化`
- 行内 `分析`
- 行内 `删除`
- 行内删除模拟持仓

### 当前文案边界

不应再在前端文档里写：

- `量化候选池`

因为当前页面已经明确展示为“实时量化股票”。

## 7. 历史回放 `/his-replay`

### 页面定位

UI 文案已经改成：

- `回放股票范围`

### 当前主要区域

1. 回放配置
2. 回放股票范围
3. 回放任务
4. 回放结论与进度
5. 成交、信号、费用统计
6. 检查点资金池视图

### 当前主要动作

- `开始回溯`
- `取消`
- `删除`

当前不应再写进文档的按钮：

- `接续`

### 当前说明文案

页面本身已经说明：

- 历史回放从统一股票池中已启用量化的股票冻结范围
- 任务内只对这批股票执行检查点

## 8. 当前共同交互模式

所有页面都遵循同一套前端模式：

1. 进入路由。
2. 读取快照接口。
3. 用户触发动作接口。
4. 动作完成后重新拉当前快照。

## 9. 当前前端与旧文档的主要差异

以下旧表述已经不适用于当前前端：

- 工作台只操作“我的关注”
- live-sim 主表叫“量化候选池”
- his-replay 支持 continue
- `/portfolio` 主接口仍是旧 `/api/v1/portfolio`

如果文案和页面实际 UI 冲突，以页面当前中文文案为准：

- 工作台：统一股票池视图
- `/live-sim`：实时量化股票
- `/his-replay`：回放股票范围

## 10. 当前隐藏页面

`/real-monitor` 的后端能力和前端页面代码仍然保留，但当前版本已经从侧边栏和工作台主线入口隐藏，不作为默认导航模块展示。
