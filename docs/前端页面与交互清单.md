### 3.3 我的关注

#### 区域能力

`我的关注` 是当前首页主视角，承接：

- 手工加股票
- 从发现股票加入的股票
- 从研究情报加入的股票
- 报价刷新
- 批量加入量化候选池
- 删除关注股票

#### 表单与按钮

| 控件/按钮 | 位置 | 功能 |
|---|---|---|
| `股票代码` 输入框 | 添加表单 | 输入要加入关注池的股票代码 |
| `添加` | 添加表单 | 将股票按 `manual` 来源加入“我的关注” |
| `↻` | 工具条 | 刷新当前关注池股票的最新价格和名称 |
| `🧪` | 工具条 | 将当前选中的关注股票批量加入量化候选池 |
| `✕` | 工具条 | 清空当前选择 |
| 表头全选勾选框 | 表格表头 | 选中或取消选中当前页股票 |
| 行内 `🧪` | 表格操作列 | 将该股票加入量化候选池 |
| 行内 `🗑` | 表格操作列 | 从“我的关注”删除该股票 |

#### 表格字段

当前字段包括：

- 代码
- 名称
- 现价
- 来源
- 状态
- 操作

状态字段当前用于表达：

- 待分析
- 最近量化信号摘要
- 最近量化价格/信号快照回写结果

### 3.4 股票分析

工作台下半区保留一个统一的股票分析模块，不再并排摆“统一结果工作台”说明卡。

#### 交互顺序

当前页面顺序已经调整为：

1. 选择分析师
2. 选择分析模式
3. 选择数据周期
4. 输入股票代码
5. 点击分析

#### 控件与按钮

| 控件/按钮 | 功能 |
|---|---|
| 分析师勾选项 | 选择参与分析的角色：技术、基本面、资金面、风险、情绪、新闻 |
| `单个分析 / 批量分析` | 切换分析模式 |
| `数据周期` | 选择分析周期，如 `1y` |
| 股票代码输入框 | 输入 A 股 / 港股 / 美股代码 |
| `分析` | 触发单股分析 |
| `批量` | 触发批量分析 |
| `清除` | 清除输入框 |
| `清空` | 清空当前结果区 |

#### 结果区

分析结果直接在模块下方展开，当前包含：

- 股票基础信息
- 关键技术指标
- 股价走势图
- 各分析师观点
- 团队讨论
- 最终决策

| `覆盖当前实时模拟账户` | 连续接续模式下控制 handoff |
| `接续后自动启动定时模拟` | 连续接续模式下自动恢复调度 |

主要按钮：

| 按钮 | 功能 |

| `取消` | 取消运行中的回放任务 |
| `删除` | 删除已完成/失败/取消的回放任务 |
